3. Главное окно GUI: чтение, отображение и фильтрация скриптов.	
   Парсер успешно подключен: происходит файлов локального репозитория, отбираются скрипты .lua, анализируются шапки с метаданным и все вместе отображаются в гуи.
   Следующий этап - настроить "поведение" отображения файлов  с помощью модели. 
   Написан прототип модели ViewModel от класса QAbstractListModel, использующий паттерн MV - Model/View в Qt, теперь происходит настраиваемое отображение скриптов в Интерфейсе - в зависимости от выбранного режима программы отображаются разные скрипты, и удобная работа с ними. В дальнейшем модель еще будет обновляться.
   Для фильтрации данных модели также был написан класс ScriptFilterModel, реализующий фильтрацию  отображаемых скриптов по нескольким выпадающим спискам и поиску.

4. Окна истории изменений
   Идет разработка окна для просмотра изменений/коммитов определенного файла. По щелчку ПКМ по скрипту открывается контекстное меню в виде таблицы, где хранятся данные о дате, авторе и пометке изменения.

5. Бэк: Конфиг для удаленного репозитория, библиотека и обертка для либгит2, функции
   Была собрана базовая GIT библиотека исходников, добавлен тестовый проект подключения libgit2.
   Создан конфиг для удаленного репозитория, содержащий ссылку, ветку и другая необходимая информация.
   Продолжается реализация обертки на Qt и С++ вокруг libgit2, которая позволяет: работать с Git‑репозиторием как с объектом, отображать состояние файлов - status() и diff (), клонировать репозиторий с удаленноой ссылки по токену или логину - clone (), подтягивать обновления с удалённого репозитория - fetch().
