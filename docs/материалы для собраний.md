# Мысли
- изначально планируем, что все основные группы - разработчики, производство и метрологи работают только на чтение. Но потом, возможно, стоит добавить возможность некоторым разработчикам или производственному персоналу добавить возможность быть редактором скриптов для тех или иных приборов (например, Андрей Загородний может редактировать скрипты только для PLS, а Андрей Банщиков только для Р44).
- скорее всего пункт выше можно будет реализовать с помощью БД.
- нужно иметь ввиду, что есть скрипты, которые охватывают два типа прибора сразу, например СК4М-18 и СК4М-50 - для них скрипты ПСИ реализованы в одном файле.
